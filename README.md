# Gwendollyn Tan - Dating Profile

A beautiful, modern dating profile website for Gwendollyn Tan, hosted on GitHub Pages.

## Features

- **Responsive Design**: Works beautifully on all devices
- **Modern UI**: Clean, elegant design inspired by portfolio sites
- **Sections Included**:
  - Hero section with introduction
  - About me section with bio
  - Hobbies & passions
  - Interests showcase
  - Contact information and form

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter & Playfair Display)
